## 功能介绍

选择一个路径，并选择需要统计的语言，点击开始，即可实现统计功能。

它可以根据不同语言，分别统计出：

- 文件数量
- 代码量
- 注释量
- 空行量

将整个代码尽可能的统计的更加精准。

目前支持的语言：

- C
- Cpp
- CSharp
- Html
- Java
- JavaScript
- Python
- Vue

更多语言持续更新中。
